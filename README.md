# line-shadow
An angular based directive to shade a div with diagonal lines
##Usage
```html
<div line-shadow
     radius="-45"
     opacity="0.2"
     color="blue"
     spread="5"
     condition="isActive"></div>
```
     
For more info visit: [this page](https://mobinni.io/2015/07/12/quickfire-shading-an-object-with-diagonal-lines-in-css/)
